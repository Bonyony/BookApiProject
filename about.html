Frankie's API Book Search

This project makes use of the Google Books API to fetch data and display it reactively. Each result includes an option to view more about the book and a link to buy said book (the link is always to a Google subsidiary). Bootstrap v5 is included to give the page a clean and clear display for the data.

Clicking the image in the top left corner will take you to my GitHub page where you can view all of my other projects!
